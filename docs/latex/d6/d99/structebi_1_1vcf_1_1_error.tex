\doxysubsection{Detailed Description}
class for VCF errors.

Child classes may be used for more specific Errors. To add another error type, follow these steps\+:
\begin{DoxyItemize}
\item predeclare class before \doxylink{classebi_1_1vcf_1_1_error_visitor}{Error\+Visitor}
\item add a class at the end of this file
\item change its name, its parent, its message and its error code
\item add a new method visit in \doxylink{classebi_1_1vcf_1_1_error_visitor}{Error\+Visitor} 
\end{DoxyItemize}
