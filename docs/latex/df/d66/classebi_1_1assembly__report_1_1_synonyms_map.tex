\doxysubsection{Detailed Description}
Mapping from every contig to all its synonyms, using an additional vector to efficiently search for repeated records in the assembly report. This vector contains the structs that associate a contig with all its synonyms. A map stores the correspondence between a name and the index its synonyms are located in the vector.
