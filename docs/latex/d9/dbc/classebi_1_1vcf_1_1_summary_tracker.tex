\doxysubsection{Detailed Description}
Class that ensures similar kind of errors are reported only once.

The intended algorithm is that errors and warnings will be printed only the first time they occur, and won\textquotesingle{}t be printed again.

The summary displays the count(number of times it occurs) of the error, and the line number of its first occurrence. We distinguish between different types of errors based on their simple error message (which contains no details). The {\ttfamily error\+\_\+order} basically maintains the order in which these errors appear for the first time.
